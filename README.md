## Getting Started
Prerequisites: Node 6 or above

You can use [nvm](https://github.com/creationix/nvm#installation) to easily switch Node versions.

### Installation

```sh
npm install
```

```sh
npm run react-dev
npm start
```

Then open [http://localhost:3000/](http://localhost:3000/) to see the app.<br>
